pl/sql编程基础

☞ pl/sql 编程的基础是块 ,通过块的组合,可以开发 (过程、函数、触发器、包)

pl/sql 变量定义的规范

  • 普通变量 v_xxx
  • 常量 c_xxxx
  • 游标 xxx_cursor
  • 异常 e_xxx

块的结构

declare
    定义部分;
begin
    执行部分;
exception
    异常部分;
end;
案例1：输出hello,world
begin
  dbms_output.put_line(‘hello,world’);
end;

为了看到输出内容，需要 set serveroutput on //打开输出选项

案例2：根据用户输入的雇员编号，显示该雇员的名字
declare
    v_ename varchar2(30);
begin
    select ename into v_ename from emp where empno=&no;
    dbms_output.put_line(‘名字是’|| v_ename);
end;
案例: ①比如在实例2中，如果输入了不存在的雇员号，应当做例外处理.
②有时出现异常，希望用另外的逻辑处理，如，如果不存在就加入编号为1 ,名字为 ‘马大哈’ 这么一个人 .
declare
    v_ename varchar2(30);
begin
    select ename into v_ename from emp where empno=&no;
    dbms_output.put_line('名字是'|| v_ename);
exception
    when no_data_found then
    dbms_output.put_line('你输入的编号没有用户,我帮你添加');
    insert into emp (empno,ename) values(14,'马大哈');
end;

案例3：编写一个过程，可以接受用户id号 , 工资 ,根据id号 去修改该雇员的薪水，如果该雇员 不存在，则给出错误提示(exception).
create or replace procedure  pro1(in_empno number, newsSal number) is
begin
  update emp set sal=newSal  where empno=in_empno;
exception
  when not_data_found then
  dbms_output.put_line(‘你的输入错误!’);
end;
