Oracle中多行子查询中使用any和all操作

sql 2014-08-10

Oracle中多行子查询中使用all和any操作:

请思考:如何显示工资比部门30的所有员工的工资高的员工的姓名?

不使用all，可以这样做：

select ename form emp where sal>(select max(sal) from emp where deptno=30)

使用all，可以这样做：

select ename form emp where sal>all(select sal from emp where deptno=30)

请思考:如何显示工资比部门30的任意一个员工的工资高的员工的姓名?

不使用any，可以这样做：

select ename form emp where sal>(select min(sal) from emp where deptno=30)

使用any，可以这样做：

select ename form emp where sal>any(select sal from emp where deptno=30)

扩展(多列子查询)：

单行子查询是指子查询只返回单行、单列的数据，多行子查询是指返回单行多列数据，都是针对单列来说的，而多列子查询则是指查询返回多个列数据的子查询语句

请思考:如何查询与smith的部门与岗位完全相同的所有雇员?

select * from emp where deptno=(select deptno from emp where ename='smith') and job=(select job from emp where ename='smith')

上面这种方法其实是错误的

正确的做法

select * from emp where (deptno,job)=(select deptno,job from emp where ename='smith')
