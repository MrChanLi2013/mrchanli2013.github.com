pl/sql入门

sql 2014-08-14

pl/sql是什么?

pl/sql (procedural language / sql) 该语句可以对数据库编程，它可以定义常量，变量..
,也有条件控制语句和循环语句.

为什么需要pl/sql

  1. 可以提高效率
  2. 可以实现模块化编程
  3. 可以减少网络传输量
  4. 安全性提高
  5. 缺点是不能移植

pl/sql编程快速入门

举一个简单案例:
编写一个存储过程，该过程可以向某表中添加记录.

create procedure p1(in_empno number,in_empname varchar2) is
begin
    insert into emp(empno,empname) value(in_empno,inempname);
end;

调用: exec p1(100,”smith”);

这里说到存储过程，那么在oracl中如何创建存储过程呢?

创建过程的语法:

create procedure 过程名(参数列表) is
begin
    //执行语句..
    //执行语句
end;

在控制台调用的语句:
exec 过程名(参数的值…)

这里需要主要的地方

  1. 在创建过程的时候 可以在 create 后 带 or replace ，这样就是说，如果该过程存在了,就替换.希望小心使用.
  2. 如果在创建过程中发生错误，可以 show error 来 查看错误信息.
