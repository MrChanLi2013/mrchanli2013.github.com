pl/sql入门

sql 2014-08-14

pl/sql是什么?

pl/sql (procedural language / sql) 该语句可以对数据库编程，它可以定义常量，变量..
,也有条件控制语句和循环语句.

为什么需要pl/sql

  1. 可以提高效率
  2. 可以实现模块化编程
  3. 可以减少网络传输量
  4. 安全性提高
  5. 缺点是不能移植

pl/sql编程快速入门

举一个简单案例:
编写一个存储过程，该过程可以向某表中添加记录.

create procedure p1(in_empno number,in_empname varchar2) is
begin
    insert into emp(empno,empname) value(in_empno,inempname);
end;

调用: exec p1(100,”smith”);
