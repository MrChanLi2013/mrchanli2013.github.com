java 2015-06-17

Hibernate中一对多注解@OneToMany

parent和child直接就属于一对多关系，一个parent可能有多个child，一个child只有一个parent：

Parent类：

@Entity
@Table(name="parent")
public class Parent {
  @Id
  @GeneratedValue
  private Integer id;

  @OneToMany(cascade = CascadeType.ALL)
  private Set<Child> children;

  @Column
  private String name;

  //getter and setter
}

Child类：

@Entity
@Table(name="child")
public class Child {
  @Id
  @GeneratedValue
  private Integer id;

  @Column
  private String name;

  //getter and setter
}
