Centos7.2 下Nginx的安装与配置

2017-08-18
Linux
约 702 字 预计阅读 2 分钟

@(Centos)[nginx]

Nginx 是一个很强大的高性能Web和反向代理服务器，它具有很多非常优越的特性，很多人都在使用它。 其实在nginx官网写的比较清楚，我在这里主要写出一些基本的用法，方便一些英文不太好的同学不踩坑。

注意我的环境是centos7

安装Nginx

  1. 创建文件/etc/yum.repos.d/nginx.repo，并输入下面内容：

    [nginx]
name=nginx repo
baseurl=http://nginx.org/packages/OS/OSRELEASE/$basearch/
gpgcheck=0
enabled=1

  2. 将OS替换成centos，OSRELEASE替换成7，就代表centos7.x。

  3. 最后yum install nginx。

Centos7.2 下mysql的安装

2017-08-18
Linux
约 473 字 预计阅读 1 分钟
  1. 使用命令yum list installed | grep mysql查看mysql是否安装
  2. 如果没有安装：先在MySQL官网中下载yum源rpm安装包：https://dev.mysql.com/downloads/repo/yum/;
  3. 下载mysql源安装包wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm
  4. 安装：yum localinstall mysql57-community-release-el7-8.noarch.rpm
  5. 将yum源安装好之后就可以使用yum install mysql-community-server安装mysql
