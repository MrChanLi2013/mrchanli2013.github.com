Oracle下常用函数

2017-11-08
数据库
约 558 字 预计阅读 2 分钟

Oracle中经常使用聚合函数，那么除了这些函数，还有哪些我们会经常用到呢？

总结如下：

  • round()：四舍五入方法，SELECT ROUND( number, [ decimal_places ] ) FROM DUAL，decimal_places 数值是按照指定的小数位进行四舍五入，小数位是从0开始计算与数组下标一样
 select round(45.55, 1) from dual; -- 输出45.60
 select round(45.21, 1) from dual; -- 输出45.20
  • to_char()转换日期
 select to_char(sysdate 'YYYY-MM-DD') from dual --转换成年-月-日的格式
  • to_char()保留两位小数补0，例如：在java中查询出的字段映射到java class类属性时需要要补0
 TO_CHAR(35.00, 'FM9999999990.00')

Centos7.2 下Nginx的安装与配置

2017-08-18
Linux
约 702 字 预计阅读 2 分钟

@(Centos)[nginx]

Nginx 是一个很强大的高性能Web和反向代理服务器，它具有很多非常优越的特性，很多人都在使用它。 其实在nginx官网写的比较清楚，我在这里主要写出一些基本的用法，方便一些英文不太好的同学不踩坑。

注意我的环境是centos7

安装Nginx

  1. 创建文件/etc/yum.repos.d/nginx.repo，并输入下面内容：

    [nginx]
name=nginx repo
baseurl=http://nginx.org/packages/OS/OSRELEASE/$basearch/
gpgcheck=0
enabled=1

  2. 将OS替换成centos，OSRELEASE替换成7，就代表centos7.x。

  3. 最后yum install nginx。
