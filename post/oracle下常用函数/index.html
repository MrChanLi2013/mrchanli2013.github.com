Oracle下常用函数

2017-11-08
数据库
约 558 字 预计阅读 2 分钟

文章目录

Oracle中经常使用聚合函数，那么除了这些函数，还有哪些我们会经常用到呢？

总结如下：

  • round()：四舍五入方法，SELECT ROUND( number, [ decimal_places ] ) FROM DUAL，decimal_places 数值是按照指定的小数位进行四舍五入，小数位是从0开始计算与数组下标一样
 select round(45.55, 1) from dual; -- 输出45.60
 select round(45.21, 1) from dual; -- 输出45.20
  • to_char()转换日期
 select to_char(sysdate 'YYYY-MM-DD') from dual --转换成年-月-日的格式
  • to_char()保留两位小数补0，例如：在java中查询出的字段映射到java class类属性时需要要补0
 TO_CHAR(35.00, 'FM9999999990.00')


  • add_months()月份函数

比如获取某个时间的上一月，add_months(date, -1)； 如果要获取几天前，则直接使用你的时间加减天数就可以了： 如：当前时间的两天前：sysdate-2

  • nvl()函数：如果第一个参数为空那么显示第二个参数的值，如果第一个参数的值不为空，则显示第一个参数本来的值。主要用于空值的处理
 select nvl(null, '无数据') from dual -- 返回 无数据
  • decode()函数：可以用来代替case when操作，我在项目中常用于除数为0的情况
decode('条件', '值1', '满足值1时返回值',..., '最后有一个默认值')

 select decode(b, 0, 0,a/b) from dual; // 如果b为0时返回0，否则返回a/b

以上就是我在项目中经常使用的函数，后续遇到了陆续添加…
