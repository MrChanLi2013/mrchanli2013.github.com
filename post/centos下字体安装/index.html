Centos7.2 下字体安装

字体的安装

已经验证.ttc，.ttf格式字体的安装。本人安装的是windows下的微软雅黑到服务器。

  1. 在/usr/share/fonts目录下建立一个子目录，例如mac：
 mkdir /usr/share/fonts/mac
  1. 将字体文件放到刚刚建立的文件夹中；
  2. 建立字体索引信息，更新字体缓存：
cd /usr/share/fonts/win
mkfontscale
mkfontdir
fc-cache
  1. 使用fc-list查看字体是否安装成功。

java中使用安装的字体

在服务器中使用fc-list查看字体可以看到下面一行：

微软雅黑,Microsoft YaHei,Microsoft YaHei Light,微软雅黑 Light:style=Light,Regular

可以看到是以逗号分隔的。 在Java中就可以使用：

Font font = new Font("微软雅黑", Font.PLAIN, 14);
Font font = new Font("Microsoft YaHei", Font.PLAIN, 14);
