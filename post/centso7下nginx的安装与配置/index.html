Centos7.2 下Nginx的安装与配置

@(Centos)[nginx]

Nginx 是一个很强大的高性能Web和反向代理服务器，它具有很多非常优越的特性，很多人都在使用它。 其实在nginx官网写的比较清楚，我在这里主要写出一些基本的用法，方便一些英文不太好的同学不踩坑。

注意我的环境是centos7

安装Nginx

  1. 创建文件/etc/yum.repos.d/nginx.repo，并输入下面内容：

    [nginx]
name=nginx repo
baseurl=http://nginx.org/packages/OS/OSRELEASE/$basearch/
gpgcheck=0
enabled=1

  2. 将OS替换成centos，OSRELEASE替换成7，就代表centos7.x。

  3. 最后yum install nginx。

配置Nginx

  • Nginx安装好之后默认配置文件为nginx.conf放在目录/usr/local/nginx/conf或者/etc/nginx或者/usr/local/etc/nginx。

  • 日志文件存放位置：/var/log/nginx。

    常用的命令

  • 启动Nginx：systemctl start nginx.service。在centos7下不再使用之前的service启动了而是使用systemctl。

  • 查看Nginx状态：systemctl status nginx.service。

  • 关闭Nginx：nginx -s stop或者nginx -s quit。

  • 重新加载Nginx配置文件：nginx -s reload。

SpringBoot下Nginx Https配置

  1. 在nginx.confhttp模块中加入以下server配置：

    http {
    ......................;

    server {
        #强制跳转至https页面
        listen      80;
        server_name    yourdomain;
        return      301 https://$server_name$request_uri;
    }

    server {
        listen 443 ssl;
        server_name yourdomain;

        ssl on;
        #https证书
        ssl_certificate 1_youdomain_bundle.crt;
        ssl_certificate_key 2_youdomain.key;

        location / {
           proxy_pass https://yourdomain:8080;
           proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
           proxy_set_header X-Forwarded-Proto $scheme;
           proxy_set_header X-Forwarded-Port $server_port;
        }

        #这儿主要防止一些域名复制或者说镜像了你的网站
        if ($host = '一些域名' ) {
           return 403;
        }
     }
}

  2. 在spring boot的application.properties文件中加入以下配置：

    server.tomcat.remote_ip_header=x-forwarded-for
server.tomcat.protocol_header=x-forwarded-proto
