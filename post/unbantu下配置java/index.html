unbantu下配置java

2016-06-12
Linux
约 14 字 预计阅读 1 分钟

文章目录

@(unbantu)[java]

    sudo gedit ~/.bashrc
    export JAVA_HOME=/usr/lib/jvm/java-7-sun
    export JRE_HOME=${JAVA_HOME}/jre
    export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
    export PATH=${JAVA_HOME}/bin:$PATH 
    source ~/.bashrc
