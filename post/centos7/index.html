Centos7.2 下svn服务器搭建

2017-08-18
Linux
约 380 字 预计阅读 1 分钟

文章目录

  1. yum install subversion安装svn
  2. 初始化仓库，mkdir -p /usr/dev/svn/svnrepos创建的该目录为仓库，然后执行svnadmin create /usr/dev/svn/svnrepos，这就完成了仓库的创建

  3. cd到该目录下ls会出现：

    README.txt conf db format hooks locks

  4. 配置相关信息，cd到conf目录，会发现下面有3个文件：

    • authz：权限控制文件
    • passwd：帐号密码文件
    • svnserve.conf ：SVN服务配置文件

  1. 添加账户密码：vi passwd在最后一行添加格式：帐号=密码，如chenli=123456

  2. 设置权限：vi authz在最后一行添加

    [/] #所有目录
chenli=rw #可读可写

  3. 修改svnserve.conf文件 ,vi svnserve.conf 打开下面的几个注释：

    • anon-access = read #匿名用户可读
    • auth-access = write #授权用户可写
    • password-db = passwd #使用哪个文件作为账号文件
    • authz-db = authz #使用哪个文件作为权限文件
    • realm = /usr/dev/svn/svnrepos # 认证空间名，版本库所在目录

  4. 启动svn服务svnserve -d -r /usr/dev/svn/svnrepos
