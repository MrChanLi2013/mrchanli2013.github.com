Centos7.2 下mysql的安装

2017-08-18
Linux
约 473 字 预计阅读 1 分钟

文章目录

  1. 使用命令yum list installed | grep mysql查看mysql是否安装
  2. 如果没有安装：先在MySQL官网中下载yum源rpm安装包：https://dev.mysql.com/downloads/repo/yum/;
  3. 下载mysql源安装包wget http://dev.mysql.com/get/mysql57-community-release-el7-8.noarch.rpm
  4. 安装：yum localinstall mysql57-community-release-el7-8.noarch.rpm
  5. 将yum源安装好之后就可以使用yum install mysql-community-server安装mysql

  1. 安装成功后启动mysqlservice mysqld start

  2. 安装成功后使用mysql会在/var/log/mysqld.log中生成一个随机密码，在重设密码的时候需要按照密码策略设置，可以详情搜索一下mysql密码策略

    grep 'temporary password' /var/log/mysqld.log
mysql -uroot -p
ALTER USER 'root'@'localhost' IDENTIFIED BY 'MyNewPass4!';

  3. 开启远程访问设置GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY '刚刚设置的密码' WITH GRANT OPTION;

  4. 重启mysqlsystemctl restart mysqld

  5. 默认配置文件路径： 配置文件：/etc/my.cnf 日志文件：/var/log//var/log/mysqld.log 服务启动脚本：/usr/lib/systemd/system/mysqld.service
    socket文件：/var/run/mysqld/mysqld.pid
